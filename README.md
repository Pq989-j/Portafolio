# 🌐 Portafolio 

Este proyecto consiste en la creación de un portafolio web responsive desarrollado únicamente con **HTML5** y **CSS3**, enfocado en presentar información personal, habilidades y proyectos de manera visual y organizada.

---

## 📋 Características

El portafolio incluye las siguientes secciones:

* 👤 **Nombre y presentación:** Identidad visual de alto impacto.
* 🧠 **Habilidades:** Listado de tecnologías y herramientas dominadas.
* 📖 **Sobre mí:** Resumen de trayectoria y objetivos profesionales.
* 📞 **Contacto:** Enlaces y formularios para comunicación directa.
* 🌍 **Idiomas:** Niveles de dominio de lenguas extranjeras.
* 💼 **Proyectos:** 3 Cards detalladas con los trabajos más destacados.

---

## 🛠️ Tecnologías utilizadas

* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilizado avanzado y personalización.

---

## 🎨 Diseño y maquetación

Todo el diseño visual fue realizado aplicando principios modernos de diseño web:

* **Flexbox:** Utilizado como motor principal para la distribución, alineación y orden de los elementos.
* **Diseño Responsive:** Adaptación total a cualquier dispositivo.
* **Cards Personalizadas:** Diseño de contenedores dinámicos para mostrar proyectos.
* **Interactividad:** Implementación de efectos *hover* y transiciones suaves.
* **Organización Modular:** Código limpio y fácil de mantener.

---

## 📁 Estructura del proyecto

```bash
portfolio/
│
├── index.html       # Estructura principal del sitio
├── style.css        # Hoja de estilos (Flexbox y diseño)
```

## 📱 Responsive Design

El sitio está diseñado con una estrategia fluida para adaptarse correctamente a:

* 💻 **Desktop:** Layout completo con aprovechamiento de espacio.
* 📑 **Tablet:** Reorganización de columnas mediante Flexbox.
* 📱 **Mobile:** Navegación vertical optimizada para pantallas pequeñas.

---

## 🚀 Objetivo del proyecto

Este proyecto fue desarrollado con el fin de fortalecer habilidades en:

* **Estructuración semántica** siguiendo estándares de HTML5.
* **Maquetación avanzada** y alineación de ejes con Flexbox.
* **Estilizado visual creativo** con CSS3.
* **Creación de interfaces** de usuario (UI) modernas y limpias.
